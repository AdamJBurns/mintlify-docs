## Planned Features

| Endpoint | Description |
|----------|-------------|
| `GET /domains` | List all verified domains |
| `POST /domains` | Add a new domain |
| `POST /crawl/start` | Start a new crawl |
| `GET /crawl/{id}/progress` | Check crawl progress |
| `GET /crawl/{id}/results` | Get crawl results |
| `POST /reports` | Generate shareable reports |
| `GET /ai-review` | Request AI-powered analysis |
